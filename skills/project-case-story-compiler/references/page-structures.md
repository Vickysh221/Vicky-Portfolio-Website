# Page Structures

## 9-page portfolio HTML structure

1. Hero
2. Context / workflow scene
3. What generic AI gets wrong
4. Reframing
5. System overview
6. Reference images as semantic evidence
7. AI’s role in the workflow
8. Current outcomes
9. Closing contribution

## Optional appendices

### Appendix A
Method layer

### Appendix B
System assets / data / cases

---

## What each page needs

Each page should contain:
- a strong title
- one core idea
- readable short copy
- one visual anchor
- one clear takeaway

Avoid page overload.

---

## Reading rhythm

Alternate between:
- text-led pages
- diagram-led pages
- image-led pages
- comparison-led pages

Do not make every page a text block.
