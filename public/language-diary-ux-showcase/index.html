Default View · New Layout
What changed: the old long page is preserved as backup, but the default entry now behaves like a case-study deck rather than a responsive dashboard wall.
Hard rules: fixed phone dimensions, floating variant titles, no right-side note blocks, and a shared bottom annotation for each spread.
Default enters the new portfolio system. The legacy page is still available as a safe fallback inside the second tab.
