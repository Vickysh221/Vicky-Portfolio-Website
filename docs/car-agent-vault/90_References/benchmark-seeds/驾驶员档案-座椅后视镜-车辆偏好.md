## Prototype 抽象

## Tesla 官方锚点
Driver Profiles：保存座椅、方向盘、后视镜以及导航/温度/驾驶偏好，可与钥匙或 Tesla Profile 关联。

## 小鹏 官方锚点
车辆控制 / 场景化设置能力广，智能座舱可自动调用多项功能。

## 理想 官方锚点
理想同学可查询更多车辆设置状态；多屏与语音跨屏交互增强。

## Prototype 启发
用户‘偏好’不应只放在设置页，也应从驾驶纠正与场景选择中学习。
