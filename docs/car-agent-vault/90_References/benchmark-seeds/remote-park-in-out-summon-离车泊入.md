# Remote Park In/Out / Summon / 离车泊入

## 能力族
泊车 / 遥控/离车泊车

## Prototype 抽象
Remote Park In/Out / Summon / 离车泊入

## Tesla 官方锚点
Summon：在私有停车场/车道上用手机让车辆前后移动或移动到用户位置。

## 小鹏 官方锚点
遥控泊车：在识别到车位后下车使用钥匙控制泊车。

## 理想 官方锚点
离车泊入：找到车位后通过 App 控制自主泊入。

## Prototype 启发
把‘人在车外但仍承担监督责任’做成单独 interaction 模式。
