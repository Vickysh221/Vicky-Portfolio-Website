# APA / Autopark / 智能泊车

## 能力族
泊车 / 自动泊车

## Prototype 抽象
APA / Autopark / 智能泊车

## Tesla 官方锚点
Autopark：支持平行/垂直车位；依赖对车位边界与障碍物识别。

## 小鹏 官方锚点
智能辅助泊车：低速搜车位后开始泊入。

## 理想 官方锚点
智能泊车：通过环视和超声波搜索车位、算路、执行泊车。

## Prototype 启发
把‘搜位—算路—泊入—暂停/恢复—退出’拆成完整状态机。
